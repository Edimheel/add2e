# Références d’audit des sorts

Ce répertoire contient les références structurées utilisées pour comparer les exports Foundry aux règles ADD2E.

Règle principale : le Manuel des joueurs AD&D 2e est la source de vérité prioritaire pour toutes les règles ADD2E.

Les fichiers de ce répertoire doivent servir à vérifier :

- le nom exact des sorts ;
- le nombre de sorts par classe et niveau ;
- les niveaux ;
- les écoles ;
- les portées ;
- les durées ;
- les temps d’incantation ;
- les composantes ;
- les composants matériels ;
- les jets de sauvegarde ;
- les effets mécaniques ;
- les descriptions réelles.

Les exports Foundry restent la référence de structure technique, mais pas la référence de règle.
