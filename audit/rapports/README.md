# Rapports d’audit des sorts

Ce répertoire contient les rapports d’audit produits par lot de sorts.

Chaque rapport doit indiquer :

- la source de référence utilisée ;
- le fichier Foundry audité ;
- les sorts attendus ;
- les sorts présents ;
- les sorts manquants ;
- les écarts de nom ;
- les écarts de composants ;
- les écarts de description ;
- les écarts de script `onUse` ;
- les écarts d’image ;
- les corrections à préparer ;
- les points à tester dans Foundry.

Les rapports ne remplacent pas les corrections : ils servent à valider le diagnostic avant modification des JSON, scripts ou images.
